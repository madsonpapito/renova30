# Prompts de Imagens para o Midjourney (Escudo de Luz)

## Estilo Visual Padrão
Adicione este sufixo a todos os prompts para garantir a consistência visual das imagens na página:

**Estilo:** `cinematic lighting, dramatic shadows, photorealistic, 8k resolution, deep burgundy and gold accents, spiritual atmosphere, emotional depth --ar 16:9 --v 6.0`

## (Opcional) Dicas de Geração

1. **Variações:** Se a imagem sair muito abstrata, adicione "photography by Emmanuel Lubezki" ou "shot on 35mm lens" no prompt.
2. **Proporção:** O `--ar 16:9` garante que a imagem fique no formato paisagem (retangular deitada), ideal para cobrir a largura da tela em sites.
3. **Cores:** Mantenha os `burgundy and gold accents` (vinho e dourado) para combinar perfeitamente com as cores configuradas no CSS da página `pagina-de-vendas-tsl.html`.
