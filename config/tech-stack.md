# Stack de Tecnologia

Stack de tecnologia recomendada para o projeto `renova-30`.

- **Frontend:** React, Vite, TypeScript, TailwindCSS
- **Backend:** Node.js, Express.js, TypeScript
- **Banco de Dados:** PostgreSQL
- **Autenticação:** JWT (JSON Web Tokens)
- **Hospedagem de Vídeo:** Vimeo ou Bunny.net
- **Gateway de Pagamento:** Kiwify (Checkout Externo)
- **Hospedagem:** Vercel (Frontend), Heroku/Render (Backend)
