# Padrões de Codificação

Este documento define os padrões de codificação para o esquadrão `renova-30`.

- **Linguagem Principal:** TypeScript
- **Framework Frontend:** React
- **Framework Backend:** Node.js (Express/Fastify)
- **Estilo:** Prettier e ESLint com configurações base.
- **Nomenclatura:** camelCase para variáveis e funções, PascalCase para componentes e classes.
